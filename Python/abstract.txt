Este proyecto tiene como objetivo el desarrollo de una página web para implementar un sistema de carsharing. El sistema de carsharing consiste en el alquiler de vehículos por periodos limitados de tiempo, estos alquileres pueden durar horas o incluso días. El objetivo de este sistema es ampliar el rubro del alquiler de automóviles permitiendo a las empresas de rentado de vehículos y a los dueños de automóviles conectarse de 	una forma más sencilla con los clientes. El sistema motivara a más personas a ser parte del servicio de carsharing, debido a la facilidad para encontrar automóviles para rentar por medio de la aplicación. Con este sistema se espera alcanzar y cubrir toda la ciudad de La Paz para posteriormente expandirse a las principales ciudades de Bolivia.  Para el desarrollo de este sistema se utilizarán las tecnologías de Python, Bootstrap y Django. Finalmente se este documento incluye un manual de usuario que facilitara el uso de la aplicación a las personas externas al proyecto. .
